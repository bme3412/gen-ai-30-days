What This Demo Teaches

This interactive demo visualizes the agent loop—the fundamental pattern that transforms a simple LLM into an autonomous agent. Unlike a chatbot that responds once and waits, an agent iterates through multiple cycles until it achieves its goal.

👁

OBSERVE

The agent gathers context: your query, previous results, and available information. This is the "perception" phase where the agent understands its current state.

🧠

THINK

The agent reasons about what to do next. It analyzes the problem, considers options, and formulates a plan. This is where the LLM's reasoning capabilities shine.

⚡

ACT

The agent executes its plan—calling tools, searching for information, or generating content. Actions change the state of the world or gather new data.

🔄

REFLECT

The agent evaluates its progress. Did the action succeed? Is more work needed? Should it try a different approach? This self-assessment enables iterative improvement.
