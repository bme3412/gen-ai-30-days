from openai import OpenAI

class Agent:
    def __init__(self):
        self.client = OpenAI()
        self.messages = []

    def _call_llm(self):
        completion = self.client.chat.completions.create(
            model="gpt-4o",
            messages=self.messages,
            temperature=0
        )
        return completion.choices[0].message.content
Agent Class: The foundation. We store an OpenAI client and a message history list. The _call_llm method sends messages to the model and returns the response.
